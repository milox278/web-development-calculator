# Kalkulator Wyceny Usług Webowych 💻

Prosta, interaktywna aplikacja webowa stworzona w celu szacowania kosztów tworzenia stron internetowych. Projekt łączy logikę biznesową z podstawowymi technologiami front-endowymi.

## 🛠️ Technologie
* HTML5 (Struktura)
* CSS3 (Stylizowanie, Flexbox)
* Vanilla JavaScript (Manipulacja DOM, obsługa zdarzeń, proste algorytmy sumujące)

## 💡 Czego się nauczyłem?
Projekt pomógł mi utrwalić wiedzę z zakresu:
* Pobierania i iterowania po elementach w DOM (`querySelectorAll`).
* Nasłuchiwania na zdarzenia (`addEventListener`).
* Dynamicznej aktualizacji interfejsu użytkownika w oparciu o akcje klienta.

Projekt jest świetnym punktem wyjścia do stworzenia wtyczki pod WordPressa w przyszłości.
